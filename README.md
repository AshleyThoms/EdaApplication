# 📊 EDA Application 📊

An interactive web application for Exploratory Data Analysis (EDA) that allows users to upload datasets and visualize insights easily.

This project was built as part of my AI & ML learning journey to practice data analysis, visualization, and building interactive Python applications using Streamlit.

---

## 🚀 Demo 🚀

Live App: https://edaapplication-ashley.streamlit.app

---

## 📌 Features 📌

- Upload and analyze CSV/Excel datasets  
- View dataset summary and structure  
- Handle missing values overview  
- Data visualization with charts and graphs  
- Interactive Streamlit interface  
- Beginner-friendly data analysis tool  

---

## 🛠️ Tech Stack 🛠️

- Streamlit  
- Pandas  
- Matplotlib  
- Seaborn  
- Openpyxl
